# Pooja Kiran Bharadwaj // ML Engineering Portfolio 👔

A high-fidelity, interactive dashboard showcasing professional expertise in **Applied Machine Learning**, **Cloud Infrastructure**, and **Mission-Critical Systems**. Designed specifically for technical recruiters and hiring managers.

## 🚀 Key Features

*   **Interactive Dashboard**: A minimalist, high-performance UI built for professional technical briefings.
*   **ML Focus**: Standardized terminology and metric-driven project highlights (e.g., 60% automation gains, 99.6% latency optimization).
*   **Procedural Audio (V24)**: A built-in, network-independent piano synthesizer developed via Web Audio API. No external dependencies, 100% stable playback.
*   **Vite Architecture**: Lightning-fast load times and optimized production builds.

## 🛠️ Technology Stack

*   **Frontend**: React.js / Vite
*   **Animation**: Framer Motion
*   **Styling**: Tailwind CSS
*   **Icons**: Lucide React
*   **Audio**: Procedural Synthesis (Web Audio API)

## 💻 Local Development

1.  **Clone Documentation**:
    ```bash
    git clone [your-repo-link]
    cd website
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

## 🌐 Deployment

This project is optimized for world-class deployment platforms:

1.  **Vercel/Netlify**: Just connect your repository and click **Deploy**. Vite settings are detected automatically.
2.  **GitHub Pages**: Add the `homepage` and `deploy` scripts to `package.json` for seamless hosting.
